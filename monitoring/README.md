# 📊 Supervision

Surveillance de l'état du serveur et des conteneurs.

| Service | Rôle |
|---|---|
| **Prometheus** | Collecte des métriques |
| **Grafana** | Tableaux de bord & alertes |
| **node-exporter / cAdvisor** | Métriques système & conteneurs |
| **Portainer** | Interface web de gestion Docker |

> 🔐 Les mots de passe (Grafana, etc.) vivent dans des `.env` **non versionnés**.
